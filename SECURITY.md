# Security Policy

## Supported Versions

Currently, only the latest major release of Tardi is supported with security updates.

| Version | Supported          |
| ------- | ------------------ |
| 1.x.x   | :white_check_mark: |
| < 1.0   | :x:                |

## Reporting a Vulnerability

Security is a top priority for us, especially because Tardi handles LLM API keys via the OS keychain and executes non-deterministic code.

If you discover a vulnerability, **do not open a public issue.** 

Please email the maintainers directly or use the GitHub Security Advisory feature to responsibly disclose the issue. We will triage and provide a response within 48 hours.
